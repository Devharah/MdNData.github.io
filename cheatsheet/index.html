1) Setting HTML.html file

< !DOCTYPE html >

The first line of the file.html must contain this line of code, it is useful to make the browser understand that the HTML language is being used and in particular HTML5 (the latest standard released).

< html >

It is used to create the structure of the html page, everything inside these tags will be interpreted as html text.

╚> lang="it"

To tell the browser what language our content is displayed in, we need to add
lang="" in the html tag : < html lang="en" > (if the content is in English).

< head >

It must be added just below the html tag, it contains the metadata of the page: technical declarations, not always actually visible on the page.

╚> < title >

It is used to give a name to our page, this name will be displayed at the top of the browser tab, and is inserted between the head tag.
